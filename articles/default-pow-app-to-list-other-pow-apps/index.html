An app for Pow that will list your other Pow apps

May 30, 2013

I use Pow to manage my Rails applications. As a consultant I tend to have quite a few Rails applications and bounce between them occasionally. Sometimes I forget what I’ve named them.

My solution was to leverage Pow’s default virtual host with a small app that dynamically lists/links my other applications. This way I can simply load http://localhost.dev in my browser and see the list of other apps, click the one I want and go on my way.

For more information see the Github repository.

If you find this useful, you might also find my Alfred Workflow for Pow handy.
